CSS

CSS语法不难，但是内容很多，所以就需要暂时参考CSS手册，必备强烈推荐。

层叠样式表（Cascading Style Sheets），与HTML一样，也是一种标记语言，其作用就是给HTML页面标签添加各种样式，定义网页的显示效果，将网页和显示样式进行分离，提高了现实功能。简单一句话就是，CSS的代码可以告诉浏览器怎么美化HTML标签的内容。

CSS入门使用

CSS代码可以卸载html标签中作为标签的属性值存在，也可以写在html网页中，作为附加代码存在，也可以单独写在css文件中通过html的link标签引入使用。

行间样式/行内样式

CSS代码可以作为在每一个标签的style属性中作为值使用。

语法格式：**<标签名 style="样式属性:属性值;样式属性:属性值...">**

开发中，针对css提供的三种引入使用方式，我们一般最常用的是外部样式，其次是内部样式，而行间样式在css使用过程中基本上是禁止使用的，而在js特效编写中，行间样式是最常用的。所以三种方法都要掌握。

CSS的基本语法

作为一门标记语言，css的语法是比较简单的，主要由三部分构成：选择符，属性和属性值。

注意，在前端领域中，有些人会觉得css负责外观样式这块，很容易存在重复代码的情况，因此就有人就基于服务端编程语法，实现了一些css的语法拓展工具如less，sass等。实现了css语法的一些拓展工具，这些工具可以让开发者在语法拓展工具之后，可以在css中实现函数，变量，表达式，循环，判断等操作。当然，即使使用了less或者sass，里面编写的代码还是属于css的原生代码，浏览器只识别css。

注意：在css中，HTML文档的标签，也叫元素。

结构描述
选择符用于查找要美化的一个或者多个html标签。
样式属性设置HTML标签的外挂效果类型，例如：长度，颜色等
样式属性值外观的具体效果描述参数。

选择符

选择符（selector），主要是为了按照规定的查看规则，告诉浏览器，我们要给哪些元素需要添加外观效果。

元素选择符

通过选择符直接找到HTML元素，添加样式。

选择符描述
Tag类型选择福，也叫元素选择符号或标签名选择符，表示通过指定元素的标签名作为选择符，给他们添加样式。
*通配选择福，表示所有html元素。
#idID选择福，表示通过给标签添加ID属性值，给唯一的ID属性值的单个元素添加样式。
.class类选择符，表示通过给元素添加class属性值对元素进行累分，给所有具有当前class属性值的标签添加样式。

关系选择符

关系选择符主要通过元素与元素之间的嵌套关系（父子关系，祖先后代关系）和并列关系（兄弟关系，同辈痛惜）来查找元素。

HTML中，元素与元素直接存在两种关系：

选择符描述
选择符1,选择符2{}群组选择符，表示同时给多个不同的选择符对应的元素添加外观效果，也叫并集选择符。
选择符A 选择符B{}包含选择符，表示给指定的（祖先）给选择符A下的（后台）选择符B对应的元素添加外观效果，也叫后代选择符。
选择符A>选择符B{}子选择符，表示给指定的（父辈）选择符A下一层的（子）选择符B对应的元素添加外观效果，也叫子选择符。
选择符A+选择符B{}相邻选择符，表示给跟着选择符A的的下一个（同辈）选择符B对应的元素添加外观效果。
选择符A～选择符B{}兄弟选择符，表示给与选择符A拥有同一个父辈的选择符B对应的元素添加外观效果，也叫同辈选择符。

伪类选择符

伪类选择符，是根据元素的位置或使用状态（交互状态）来查找某一类元素。

选择符描述
选择符:link设置超链接未被访问前的样式。
选择符:visited设置超链接在其链接地址已被访问过时的样式。
选择符:hover设置元素在其鼠标悬停时的样式。
选择符:active设置元素在被用户激活（鼠标点击时与释放直接发生的时间）时的样式。
选择符:focus设置元素在成为输入焦点（表单输入框，或者具有editcontent的元素）时的样式。
选择符:first-child匹配父元素的第一个子选择符。
选择符:last-child匹配父元素的最后一个子选择符。
选择符:nth-child(n)匹配父元素的第n个子选择符。
选择符:checked匹配处于选中状态的选择符。（用于 input type为radio和checkbox时）
选择符:disabled匹配用户界面上处于禁用状态的选择符。（常用语表单项）

CSS样式优先级

可以先看一下这个文档

css在使用过程中，因为很多的选择符都具有批量查找的特点，所以很容易出现样式污染的问题。样式污染出现的原因是因为css代码编写的过程中，因为引入方式，书写先后顺序以及选择符的优先级等问题导致的。

CSS样式的优先级

针对css中的代码，使用不同的引入方式，也会存在优先级的问题。三种引入方式中，行内样式的权重最高，内部样式和外部样式的权重一样，所以只有css不是行内样式，不管是外部还是内不样式的使用过程中，写在后面的css代码，会覆盖前面写的css代码。其中，如果在样式属性值后面加上!important,则表示无视权重（权重最高），优先使用当前属性值外观展示。

在css使用过程中，浏览器本身辉与html元素的外观实际上预设了一些默认样式的。这些样式存在了user agent stylesheet。所以实际上，user agent stylesheet提供的样式才是权重最低的。所以也是因为浏览器本身对于不同的HTML元素内部预设了一些默认外观效果，所以w3c又针对这两种情况，又提供了两个纯净标签，是几乎没有任何内置样式的：div span。

div标签代表网页的一个板块，自带换行效果，span标签代码的是文本的一部分内容，所以不会自带换行，也没有任何外观样式。div+span+css往往用户网页的排版布局。

选择符的优先级

当css代码采用外部样式或内部样式时，针对同一个元素，使用不同的选择符，也会存在优先级的比较，容易导致样式污染的问题。
